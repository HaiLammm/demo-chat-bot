# Kỹ năng Quản lý Package (Package Management)
- **Ưu tiên:** Sử dụng Homebrew trên Linux/Ubuntu để cài đặt các công cụ hệ thống (system tools) và các ngôn ngữ lập trình.
- **Thực hành:** Sử dụng `brew install [package]` thay vì `sudo apt install [package]` trừ khi không còn lựa chọn nào khác.

# Công cụ và Thực hành chung
- **Version Control:** Kỹ năng sử dụng Git (rebase, merge, pull requests) là bắt buộc.
    - **DevOps/Môi trường:** Hiểu biết về việc triển khai (deployment) ứng dụng Next.js trên Vercel/Netlify hoặc triển khai Python Backend bằng Docker.
